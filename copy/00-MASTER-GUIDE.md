## SEO Strategy Summary

### Target Audience Personas

1. **The Procurement Officer** — Evaluating vendors for government or private sector tenders; cares about compliance, B-BBEE, and registration documents.
2. **The Project Manager** — Looking for a reliable subcontractor for a specific scope of work; cares about track record, capabilities, and references.
3. **The Network Operator** — Seeking telecoms installation partners for fibre rollout projects; cares about certifications, operator experience, and quality.
4. **The Property/Facility Manager** — Needing vegetation management or building maintenance; cares about reliability and responsiveness.

### Primary Keyword Clusters

| Cluster | Target Keywords |
|---------|----------------|
| **Brand** | Thoyiwa Global Technology, thoyiwaglobaltech |
| **Construction** | construction company Krugersdorp, construction services Gauteng, CIDB registered contractor, trenching and excavation services |
| **Telecommunications** | fibre installation South Africa, FTTH contractor Gauteng, cable jointing services, fibre splicing contractor, telecommunications infrastructure company |
| **Vegetation** | vegetation management South Africa, tree pruning services Gauteng, grass cutting contractor, Eskom vegetation management |
| **B-BBEE / Transformation** | female-owned construction company, youth-owned business South Africa, B-BBEE Level 1 construction company |
| **Location** | infrastructure company Krugersdorp, contractor Johannesburg, construction services Pretoria |

### On-Page SEO Checklist (Per Page)

- [x] Unique, keyword-rich `<title>` tag (50–60 characters ideal)
- [x] Compelling `<meta description>` (150–160 characters ideal)
- [x] Single `<h1>` per page
- [x] Logical heading hierarchy (H1 → H2 → H3)
- [x] Primary keywords in H1 and first 100 words
- [x] Internal linking between pages
- [x] Structured data markup recommended (FAQPage, LocalBusiness, Organization)
- [x] Image alt text suggestions included
- [x] CTAs on every page section

### Technical SEO Recommendations

| Item | Recommendation |
|------|----------------|
| **Site Speed** | Optimise images (WebP format), minify CSS/JS, enable caching |
| **Mobile** | Fully responsive design, mobile-first approach |
| **SSL** | HTTPS required across entire site |
| **Sitemap** | Generate XML sitemap; submit to Google Search Console |
| **Robots.txt** | Allow crawling of all public pages |
| **Schema Markup** | LocalBusiness, Organization, FAQPage, BreadcrumbList |
| **Google Business** | Claim and optimise Google Business Profile for Krugersdorp location |
| **Analytics** | Google Analytics 4 + Google Search Console |

## Notes for Developers

1. **SEO Metadata**: Each page file contains a metadata table — implement as `<title>`, `<meta>`, and OG tags.
2. **Structured Data**: Implement JSON-LD schema markup as recommended (especially FAQPage for FAQ).
3. **Forms**: Contact and Support forms should POST to a backend handler with email notification.
4. **Image Placeholders**: Image suggestions are provided per section — source or shoot appropriate imagery.
5. **Responsive Design**: All sections are designed for mobile-first implementation.
6. **Analytics**: Track form submissions, CTA clicks, and page engagement as conversion events.
7. **Accessibility**: Ensure WCAG 2.1 AA compliance — alt text, colour contrast, keyboard navigation.
